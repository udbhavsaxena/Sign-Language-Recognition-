# Sign Language Recognition

Recognize a small set of sign-language gestures using MediaPipe keypoints and an LSTM classifier.

## Overview

This project uses MediaPipe Holistic to extract pose, face, and hand landmarks from video frames, then trains an LSTM sequence classifier over 30-frame keypoint windows. It was created as an educational deep learning project and includes the original report and notebook plus a reusable script extracted from the notebook workflow.

## Features

- MediaPipe landmark extraction
- Keypoint sequence dataset loading from `.npy` files
- LSTM-based action recognition model
- Webcam inference loop
- Original project report in `Report.pdf`

## Tech Stack

- Python
- TensorFlow / Keras
- MediaPipe
- OpenCV
- NumPy
- scikit-learn
- Jupyter Notebook

## Architecture / Workflow

```text
webcam/video frames -> MediaPipe landmarks -> 30-frame keypoint sequence -> LSTM classifier -> gesture label
```

## Setup Instructions

### Prerequisites

- Python 3.9+
- Webcam access for real-time inference
- Pre-extracted `MP5_Data/` keypoint dataset or a trained model file

### Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Environment Variables

No environment variables are required.

### Running the Project

Train from keypoint arrays:

```bash
python src/sign_language_recognition.py train --data-dir MP5_Data --output models/projsen.h5
```

Run webcam inference:

```bash
python src/sign_language_recognition.py webcam --model models/projsen.h5
```
